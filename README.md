# 📋 Enrollment Form — Estrelas do Amanhã

A responsive enrollment form for a children's school, built with semantic HTML and modular CSS.

## ✨ Features

- 📝 Multi-section form layout (child info, medical, address, guardian, options)
- 🔘 Custom styled radio buttons and checkboxes
- 📎 File upload field (birth certificate)
- ☀️ Shift and sport activity selection
- ⚠️ Inline email validation feedback
- 🖥️ Two-column responsive layout

## 🛠️ Tech Stack

- HTML5
- CSS3 (modular — fields, forms, layout, global)
