# 📋 Formulário de Matrícula — Estrelas do Amanhã

Formulário de matrícula responsivo para uma escola infantil, desenvolvido com HTML semântico e CSS modular.

## ✨ Funcionalidades

- 📝 Layout com múltiplas seções (dados da criança, informações médicas, endereço, responsável e opções)
- 🔘 Radio buttons e checkboxes estilizados
- 📎 Campo de upload de arquivo (certidão de nascimento)
- ☀️ Seleção de turno e atividade esportiva
- ⚠️ Validação de e-mail com feedback inline
- 🖥️ Layout responsivo em duas colunas

## 🛠️ Tecnologias

- HTML5
- CSS3 (modular — fields, forms, layout, global)
